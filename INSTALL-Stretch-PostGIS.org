** Dependencies and clients

We first check or install required libraries: GDAL (), GEOS () and
Proj4 ():

#+BEGIN_SRC sh
  sudo apt install gdal-bin libgdal-dev libgeos-dev proj-bin libproj-dev
#+END_SRC
